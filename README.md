# WALKWAVE | Premium Footwear Showcase

Welcome to **WALKWAVE**, a sophisticated and elegant footwear showcase web application. This project highlights a premium collection of shoes across various distinct categories, featuring a clean, airy, and ocean-inspired minimalist user interface.

## 🌟 Live Demo
The project is live and deployed via GitHub Pages:
👉 **[View the Live Site Here!](https://Dassuparna-ops.github.io/show_cave/)**

## ✨ Features
* **Light & Airy Design**: A fresh, ocean-inspired minimalist aesthetic utilizing crisp whites, deep charcoal text, and beautiful teal/blue gradients.
* **Fully Responsive**: Optimized for all devices, including a custom off-canvas hamburger navigation menu for mobile screens.
* **Smooth Animations**: Integrated micro-interactions such as subtle card lift hover effects, scaling transitions, and smooth page load fade-ins.
* **Premium Collections**: Showcases five distinct product lines:
  * Heritage Boots
  * Metro Kicks
  * Apex High-Tops
  * Coastal & Summer
  * Cloud Comfort Slippers
* **Modern Typography**: Utilizes the highly legible and modern *Plus Jakarta Sans* font family for a premium reading experience.

## 🛠️ Built With
* **HTML5**: Semantic and structural layout.
* **CSS3**: Advanced styling, CSS variables, flexbox/grid layouts, and elegant drop shadows.
* **Vanilla JavaScript**: Navigation toggles, scroll animations, and dynamic layout interactions.

## 🚀 Getting Started

If you want to run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dassuparna-ops/show_cave.git
   ```
2. **Open the project directory:**
   ```bash
   cd show_cave
   ```
3. **Launch the site:**
   Since this is a static website, you can simply open `index.html` in your favorite web browser. Alternatively, you can use a local development server like Live Server (for VS Code) or Python:
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000` in your browser.
